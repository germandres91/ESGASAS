Exploración Minera

Detección de vetas minerales y estructuras geológicas mediante análisis de resistividad

Descripción del Proyecto

Objetivo

Identificación y caracterización de estructuras geológicas favorables para la mineralización mediante técnicas geofísicas avanzadas.

Ubicación

Zona minera de Boyacá, Colombia

Duración

3 meses de trabajo de campo y análisis

Equipo Utilizado

  • • Sistema de tomografía eléctrica 2D/3D
  • • Electrodos de acero inoxidable
  • • Cables de conexión especializados
  • • Baterías de alta capacidad

Metodología

1

Reconocimiento Geológico

Mapeo de estructuras y muestreo de afloramientos

2

Diseño de Perfiles

Configuración de líneas de tomografía eléctrica

3

Adquisición de Datos

Mediciones de resistividad en campo

4

Procesamiento e Interpretación

Análisis de datos y modelado geológico

Resultados Obtenidos

15

Perfiles de Tomografía

3

Zonas Anómalas Identificadas

85%

Precisión en la Detección

Especificaciones Técnicas

Configuración de Equipos

  • • Electrodos: 120 unidades
  • • Espaciado: 5 metros
  • • Profundidad máxima: 200 metros
  • • Resolución: 1 metro

Parámetros de Medición

  • • Voltaje: 400V máximo
  • • Corriente: 1A máximo
  • • Frecuencia: 0.1 Hz
  • • Tiempo de integración: 30 segundos

Análisis de Datos

Software Utilizado

  • • RES2DINV para inversión 2D
  • • RES3DINV para inversión 3D
  • • Surfer para visualización
  • • AutoCAD para planos finales

Criterios de Interpretación

  • • Anomalías de baja resistividad
  • • Estructuras lineales
  • • Contactos geológicos
  • • Zonas de fracturación

Recomendaciones

Próximos Pasos

  • • Perforación de confirmación en zonas anómalas
  • • Análisis geoquímico de muestras
  • • Evaluación económica preliminar
  • • Plan de exploración detallada

Beneficios Obtenidos

  • • Reducción del riesgo exploratorio
  • • Optimización de recursos
  • • Información geológica detallada
  • • Base para decisiones de inversión
